# Final Project: Event Sharing Website for Friends

## Repository name: fluffy-octo-waddle

## Team Composition: Solo

## Problem Statement

- I don't know what my friends are up to.
- I don't want the pressure of formally inviting friends and waiting for their response.
- I want to search and grab for the occasions I want without the pressure of RSVP'ing.

## Proposed Solution

A site in which users can log in and post or delete ics files. Files should change to "past" status when the event time has passed. ICS files should be sorted in chronological order.

## Features

### MVP

- Ability to register as a new user
- Ability to log in
- Ability to upload an ics file
- Ability to view all uploaded ics files

Users can create accounts, log in, upload ics files, and view all ics files uploaded by other users.

### Nice to have

- Ability to delete account
- Multiple instances with admins and separate user groups who can only see that set of events.
- Past events are archived and only viewable under a special filter
- Ability to delete events that the user has uploaded (not other's events)
- Ability to RSVP to events
- Ability to see which users have RSVP'd to events
